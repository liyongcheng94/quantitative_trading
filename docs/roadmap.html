01
PHASE 01 · FOUNDATION
打地基·金融通识与数据工具
⏱ 1–2 周
学习目标

理解市场基本规则，掌握核心数据处理工具。

📊

金融入门

通识

理解股票、ETF 等基本概念，弄懂 K 线、成交量、市盈率（PE）等术语。重点理解 A 股的 T+1 交易制度和涨跌停板规则。

🐍

Python 工具强化

技能

你肯定熟悉 Pandas 和 NumPy，现在可以重点关注其在金融数据分析中的应用。关键是习惯用"向量化运算"替代 for 循环来处理大规模时间序列数据。

工具栈 Pandas NumPy 向量化运算 K 线 PE T+1 涨跌停
↓
02
PHASE 02 · BUILD
建系统·回测框架与经典策略
⏱ 3–5 周
学习目标

搭建第一个回测系统，实现并验证经典策略。

🧰

选择回测框架

工具

从 Backtrader 或 Zipline（或其国内版本 Rqalpha）入手。它们能让你专注于策略逻辑。更现代的 VectorBT 以速度快著称，也可以了解。

📈

实现经典策略

实战

动手实现双均线策略、海龟交易法则和网格交易策略。目标不是靠它们赚钱，而是跑通"数据输入 → 策略信号 → 回测输出 → 绩效评估"的完整流程。

工具栈 Backtrader Zipline Rqalpha VectorBT 双均线 海龟 网格
↓
03
PHASE 03 · AI / ML
玩 AI·AI/ML 在量化中的应用
⏱ 6–12 周
学习目标

将 AI/ML 技能应用于金融场景，构建预测模型。

🌱

入门：传统 ML

入门

从 Scikit-learn 入手，将随机森林、XGBoost 等模型应用于因子合成或涨跌预测。

🧠

进阶：DL + RL

进阶

使用 TensorFlow / PyTorch，探索 LSTM 等模型进行时间序列预测。进阶可了解强化学习在优化交易参数上的应用。

🤖

前沿：AI Agent

前沿

了解如何构建一个由"AI 情报官"、"AI 策略员"等多角色协作的自动化系统。

工具栈 Scikit-learn XGBoost TensorFlow PyTorch LSTM 强化学习 AI Agent
↓
04
PHASE 04 · LIVE
上战场·模拟盘与实盘
⏱ 持续进行
学习目标

连接实盘，用真金白银验证并迭代策略。

🧪

模拟盘验证

验证

在 聚宽（JoinQuant）等平台用模拟盘检验策略，这是实战前的重要缓冲。

⚙️

连接实盘

实战

资金达到门槛后，可开通 miniQMT 或 Ptrade 进行实盘交易。

🛡️

风控第一

纪律

严格遵守止盈止损纪律。永远先跑通回测 → 模拟盘 → 小资金实盘的流程。

工具栈 聚宽 JoinQuant miniQMT Ptrade 止盈止损 仓位管理
⚠️ FUNDAMENTAL RULE

风控第一，永远第一

策略可以试错，本金不能熔断。
不管你在哪个阶段，都请遵守这条不可逾越的流水线：

回测 → 模拟盘 → 小资金实盘 → 逐步加仓
