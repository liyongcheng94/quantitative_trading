01
打地基
金融通识与数据工具
⏱ 1–2 周
目标 理解市场基本规则，掌握核心数据处理工具。
  • 金融入门：股票、ETF、K 线、成交量、PE
  • A 股特色：T+1 制度、涨跌停板
  • Python 强化：Pandas / NumPy 金融场景应用
  • 核心习惯：向量化运算替代 for 循环
Pandas NumPy 向量化 K 线 T+1
02
建系统
回测框架与经典策略
⏱ 3–5 周
目标 搭建第一个回测系统，实现并验证经典策略。
  • 回测框架：Backtrader / Zipline / Rqalpha
  • 速度优先：VectorBT 向量化回测
  • 经典策略：双均线、海龟、网格
  • 跑通闭环：数据 → 信号 → 回测 → 绩效
Backtrader Zipline Rqalpha VectorBT 双均线 海龟
03
玩 AI
AI/ML 在量化中的应用
⏱ 6–12 周
目标 将 AI/ML 技能应用于金融场景，构建预测模型。
  • 传统 ML：Scikit-learn、随机森林、XGBoost
  • 应用场景：因子合成、涨跌预测
  • 深度学习：TF / PyTorch、LSTM 时序预测
  • 前沿探索：强化学习、多 Agent 协作
Scikit-learn XGBoost PyTorch LSTM RL Agent
04
上战场
模拟盘与实盘
⏱ 持续进行
目标 连接实盘，用真金白银验证并迭代策略。
  • 模拟盘：聚宽 JoinQuant 检验
  • 实盘通道：miniQMT、Ptrade
  • 风控第一：止盈止损纪律
  • 流程红线：回测 → 模拟 → 小资金
聚宽 miniQMT Ptrade 止盈止损 仓位管理
⚠️ 风控红线 无论哪个阶段，都必须按顺序走：
回测 → 模拟盘 → 小资金实盘 → 逐步加仓
